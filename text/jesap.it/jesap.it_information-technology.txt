area it
 
 
 


L’Area IT opera sia internamente all’associazione, sviluppando applicativi di supporto per la digitalizzazione della stessa, sia esternamente, offrendo servizi IT per imprese terze.
 
 
 


 
 


Le attività svolte dall’area IT sono divise in interne ed esterne.
Le prime sono riguardano:

Software per il monitoraggio delle attività interne
Dashboard interattive
Supporto alla digitalizzazione dell’associazione

Le attività esterne invece sono incentrate sulla realizzazione di servizi di consulenza in ambito IT:

Software gestionali

CRM
ERM
Storage management



 

Creazione di Web App
Dashboard interattive per il supporto ai servizi delle altre aree
Applicativi di Machine Learning (Ottimizzazione SEO)
